Detailed Description

Definición de las funciones de transmisión LoRa (SX1262/RadioLib) para el nodo de la mascota.

Proporciona:

  • LORA_begin(freqMHz): inicialización del módem y parámetros físicos.
  • LORA_startTx(buf,len): inicio de transmisión asíncrona.
  • LORA_isTxDone()/LORA_lastState(): consulta del estado de TX.
  • LORA_finishTx(): cierre explícito de la transmisión.
Warning
Comprobar límites de duty-cycle según ETSI EN 300 220 (EU 868 MHz).

Inicializa el SX1262 con parámetros LoRa por defecto.

Parámetros usados en radio.begin():

  • BW = 125.0 kHz
  • SF = 9
  • CR = 7 (equivale a 4/7)
  • syncWord = 0x12 (privado)
  • power = 14 dBm
  • preambleLen = 8
  • tcxoVoltage = 0 (sin TCXO)
  • useRegulatorLDO = false
