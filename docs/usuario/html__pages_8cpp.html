Detailed Description

Implementación de la generación de páginas HTML dinámicas del portal web.

Este módulo genera y sirve las páginas HTML necesarias para la interfaz de usuario del nodo receptor, incluyendo:

  • Listado de redes WiFi disponibles.
  • Confirmación de credenciales almacenadas.
  • Visualización de coordenadas GNSS recibidas.

Las páginas se cargan desde el sistema de ficheros LittleFS y se envían mediante el servidor web integrado (WebServer).
