Detailed Description

Implementación de la gestión de conexión WiFi y configuración de red.

Este módulo controla la conexión del nodo a una red WiFi y el almacenamiento de credenciales en memoria persistente (LittleFS). Permite:

  • Cargar y guardar SSID/contraseña.
  • Intentar conexión en modo estación (STA).
  • Activar un punto de acceso (AP) para configuración manual.
  • Manejar el formulario HTML de envío de credenciales.

Integra feedback visual mediante la pantalla LCD y coordinación con el servidor WebServer.

Note
Compatible con el stack WiFi del núcleo RP2040 de Raspberry Pi Pico W.
